Comandos git para manipular arquivos

Após baixarmos o clone no computador temos de subir as modificações feitas, para isso usamos os comandos que aprendemos na aula3

git add nome_do_arquivo ou git add . para subir todos os arquivos de uma vez

git commit -m"texo_do_que_foi_modificado" para fazer o commit local

git push -u oringin master para subir os commits para o repositório remoto no Github

Atentar para sempre antes de fazer o push verificar no Source Control se o que modificamos foi realmente o desejado

e fazer o pull para baixar todas as alterações que já estão no repositório remoto

git pull origin master

Atentar também para o nome da branch, no caso deste repositório criado no Github é "main", portanto o código seria git push -u oringin main
